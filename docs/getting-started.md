# Getting Started

1. Clone the repository.
2. Install the Python dependencies required by the IoT, sensor, data collection,
   maintenance, and notification modules.
3. Configure device, sensor, and notification settings for the target elevator
   environment.
4. Run the main platform script:

```bash
python autonomous_elevator_maintenance.py
```

The platform monitors elevator data, processes the collected signals, identifies
maintenance needs, and schedules maintenance tasks.
